The Independent and Open Source
Standard for Avatars.

It's time to move away from the bloat of Gravatar and similar centralized solutions. With ODPX, you get a lightweight, open-source, and self-hostable alternative that puts control back in the hands of users and service providers.

Get started


Simple Setup

ODPX is made to be low-cost, low-maintenance, and low-stress with easy setup & updates.


Independence

Anyone can host their own ODPX service. Forget relying on a single centralized provider.


Dynamic Resizing

Specify the size and format you need when requesting an avatar.


Secure & Private

No unnecessary logging. Emails are always hashed in transit.

No Centralized Control

Unlike Gravatar, which is controlled by a single provider, emblm allows you to run your own ODPX service. You control the data, the privacy, and the availability. By using DNS-based delegation via TXT records, ODPX doesn't require a central instance for coordination.

Easy to Integrate

Using ODPX is as easy as self-hosting your own instance and/or using the standardized lookup-flow and API to request avatars from other instances.
